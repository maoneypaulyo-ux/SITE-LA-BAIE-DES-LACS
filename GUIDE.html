Étape 4 : Google Maps

  1. Allez sur Google Maps
  2. Cherchez votre hôtel
  3. Cliquez "Partager" → "Intégrer une carte"
  4. Copiez le code <iframe>
  5. Collez-le dans pages/contact.html (ligne ~200)

Personnaliser votre site

Modifier le nom de l'hôtel

Recherchez dans tous les fichiers :

Notre Hôtel

Remplacez par le nom de votre établissement

Corriger les prix des chambres

Dans pages/rooms.html, recherchez les prix et modifiez :

À partir de 50€/nuit → À partir de VOTRE_PRIX€/nuit

Modifier les descriptions

Chaque section avec data-fr="..." et data-en="..." est multilingue.

Exemple : Cherchez "Confort, qualité et hospitalité" pour changer le slogan

Changer les couleurs

Ouvrez css/style.css et modifiez les variables au début :

:root { --primary-color: #2c3e50; /* Bleu foncé - à changer */ --secondary-color: #e74c3c; /* Rouge - à changer */ --accent-color: #3498db; /* Bleu clair - à changer */ }

Fonctionnalités principales

Changement de langue

Le bouton EN/FR en haut à droite change la langue automatiquement.

Le choix est mémorisé dans le navigateur de l'utilisateur.

Bouton WhatsApp

Un bouton vert flottant s'affiche en bas à droite sur tous les appareils.

Assurez-vous d'avoir configuré votre numéro WhatsApp !

Responsive Design

Le site s'adapte automatiquement à :

  • Téléphones (320px et +)
  • Tablettes (768px et +)
  • Ordinateurs (1024px et +)

Formulaire de contact

Important : Le formulaire affiche un message de succès pour l'instant. Pour recevoir les emails, vous devez configurer un service comme Formspree (gratuit) ou EmailJS.

Questions fréquentes

Q : Comment j'ajoute une nouvelle chambre ?

Ouvrez pages/rooms.html, copiez un bloc .room-card entier et modifiez les informations.

Q : Où héberger le site gratuitement ?

  • GitHub Pages (gratuit, domaine yourname.github.io)
  • Netlify (gratuit avec domaine perso optionnel)
  • Vercel (gratuit, très rapide)

Q : Comment j'achète un nom de domaine ?

  • monhotel.com chez Google Domains (~10-15€/an)
  • monhotel.fr chez OVH (~5-8€/an)
  • Namecheap - moins cher

Q : Le formulaire de contact ne marche pas de quel côté ?

Vous devez configurer un service backend comme :

  • Formspree (gratuit, facile)
  • EmailJS (gratuit, JavaScript)
  • JotForm (gratuit, interface visuelle)
